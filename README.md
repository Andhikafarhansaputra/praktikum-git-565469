## Tugas 1: Inisialisasi & Riwayat Commit

Tahap pertama adalah repository diinisialisasi dan dihubungkan ke GitHub 
- Dilakukan beberapa commit awal menggunakan standar **Conventional Commits**:
  - `feat:` → fitur baru  
  - `fix:` → perbaikan bug  
  - `style:` → perubahan tampilan  

- Menambahkan file `.gitignore` untuk menghindari upload file tidak penting seperti:
  - `.DS_Store`
  - `node_modules/`
